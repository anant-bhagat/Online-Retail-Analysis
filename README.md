# Online Retail Data Analysis Project

## Project Overview

This project focuses on performing end-to-end exploratory data analysis (EDA) on an online retail transactional dataset using Python.

The objective was to understand:

* Revenue growth trends
* Customer purchasing behavior
* Customer retention and repeat purchases
* Product-level performance
* Country-wise revenue contribution
* Business opportunities and risks

This project simulates how a data analyst would evaluate business performance using raw transactional data.

---

# Business Problem

An online retail company generates thousands of transactions across multiple countries.

The business wants to understand:

* Which customers contribute most revenue
* Which products drive performance
* Whether repeat customers impact revenue
* Which countries are most valuable
* How customer retention influences business growth

---

# Dataset Information

**Dataset Name:** Online Retail Dataset

The dataset contains real transactional retail data including:

| Column      | Description                |
| ----------- | -------------------------- |
| InvoiceNo   | Unique transaction ID      |
| StockCode   | Product identifier         |
| Description | Product name               |
| Quantity    | Units purchased            |
| InvoiceDate | Transaction timestamp      |
| UnitPrice   | Price per product          |
| CustomerID  | Unique customer identifier |
| Country     | Customer country           |
